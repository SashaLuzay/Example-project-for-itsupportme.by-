# Example-project-for-itsupportme.by-
                       Простой пример на микроконтроллере STM32F103C8T6
В примере были задействованы :UART , I2C(для примера был подключен дисплей 1602)  , 2 канала ШИМ на выводах A0 , A1.
 Команды принимаемые по UART обрабатывались в прерывании , отправлялись обратно , изменяли состояние светодиода и текст выводимый на дисплей.
